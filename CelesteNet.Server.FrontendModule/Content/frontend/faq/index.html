CelesteNet FAQ
Frequently Asked Questions about CelesteNet.
< Back to main page

This site aims to answer questions most commonly asked on CelesteNet and to act as a general guide on how to use it.
If you have any other questions, the quickest way to get them answered is by joining the official Celeste Discord server and asking in the #celestenet channel!

#1 How do I get CelesteNet?

For this, please follow the instructions on the main site.

#2 How do I join CelesteNet?

Once you've installed the mod and opened the game, go into Mod Options and find CelesteNet's settings section. Flip the "Connected" toggle. That's all!
If you run into any issues not addressed on this page, check the last entry on where to contact us or get help.

#3 Are there rules? Can I get banned?

Yes, chat is moderated and should be SFW. There's a dedicated rules page, please read it before joining. The rules are modeled after the rules of the official Celeste discord.
Usually we will issue at least one warning kick before we ban, unless there's very obvious trolling going on. Kicks and bans always have a reason attached to them.

#4 How do I register? How do I get my profile picture in-game?

Registered accounts are created by linking a Discord account. Currently there is no other way. You can always join with a Guest name, unless Lockdown is in effect. See #5 below.
Registration is done on the main site in the box titled "Your Profile". It is recommended you do this in a browser on the same device that you're playing CelesteNet on.
Once you've registered, you'll need to click "Send to client" in the "Your Profile" box, or manually set your key in-game. Your key kind of acts like a passphrase to identify you to the server, so keep it safe.
Some browsers' security features might block the website from communicating with the game client.

After you've set your key in-game, you will have to reconnect to start using the new name/profile.

Technical info:
The registration uses the OAuth protocol, similar to how you can sign up to services using your Google or Facebook account. CelesteNet will get some of your discord profile information, including your account username, discriminator (#xxxx) and profile picture.
Your name and profile picture on CelesteNet do not sync automatically - if you changed them on Discord you will have to "link" your profile again. There is no need to "revoke" your key for this however, it can stay the same.

#5 Why do I get the "Server doesn't allow anonymous guests" warning?

This means the server has been put into "lockdown" mode, meaning only registered players can join. See question #4 above on how to register.

#6 Are there any differences between Guests and registered users?

There are some differences, the most obvious one is that Guests don't have profile pictures.
Some more technical limitations exist, because the server will not store certain data about Guests. This means you cannot use /cc, /gc, /w, /tpon, /tpoff to turn on/off auto-channel-chat, whispers and teleports.
This only applies to the "toggle" commands when these are send by themselves. Guest can use /cc [...] and /w [...] and similar commands to send channel chat and whisper messages.

#7 How do I chat? How does chat work?

With default settings, press T on your keyboard to open chat.
Send /help to learn about commands you can use. All commands start with /.

By default you will be in the channel "main" like most other players. For more information about channels, check entry #9 below.

#8 Where is the server, how many servers are there?

There is only one official CelesteNet server, the same server that this site is hosted on. It is located in Germany, hosted by jade with Hetzner.
If you want to play privately with a smaller group of people, you most likely want to join a private channel rather than host your own server. See entry #9 about channels.

#9 How do I create and use channels?

By default you will be in the channel "main" like most other players.
Press the Tab key to see the player list to see your channel.

You can create a new channel or join an existing one by sending the command /join [channelname] in chat, putting in the channel name – if it does not yet exist, it will be created.

You can create a private channel by prefixing the channel name with a ! like /join !secretname - keep in mind that the "secret" name could be guessed by others if it is too simple.

Please note that at the moment channel names are case-sensitive! So joining !Hello creates a different channel than !hello. This will probably be changed soon-ish.

#10 Can I play custom maps with my friends?

Yes! All maps should work, whether they're the "vanilla" chapters or modded.
Simply being in the same level (and in the same CelesteNet channel!) as another player should let you see them and play with them.

#11 I keep accidentally grabbing/throwing people! What are "Interactions"?

Interactions are a feature that let you grab other players and throw them similar to Theo crystals. You can also bounce on players' heads.
However this feature can be a little buggy and lead to accidental grabs and throws, so we recommend to set "Interactions" to "Off" in the CelesteNet Mod Options.
(Also, if you've been thrown "into" solid tiles or got otherwise stuck due to interactions and can't even open the Pause menu, you should use /tp ... or the game's Debug map accessed with F6 to teleport yourself out)

#12 How do I race on CelesteNet? Are there multiplayer game modes?

CelesteNet itself does not provide any functionality for "racing" others.
The main mod implementing such features is Head 2 Head (GameBanana). Check the mod's page for a full description of its features.

Other popular multiplayer mods include:

  • Madhunt - a mod where you play a form of "Badeline tag"/hide & seek in arena maps
  • Heart Wars - a "capture the flag" kind of game mode
  • Partnership - a co-op map for two players
  • ...and more? If you made or know of a mod that should be listed here, let us know!

#13 Can I use other mods, assist mode, cheat mode or variant mode?

Yes!
Generally all of these are permitted. What's not allowed is using these to cheat in competitive contexts like racing or Madhunt.
Also, you shouldn't excessively use invincibility, infinite dashing and other gameplay changes to harass others or to prevent them from being able to play and enjoy the game.

#14 How do I change my character’s hair and appearance? Skin mods?

Celestenet itself only synchronizes hair length and hair colors properly, in addition to "vanilla" skin mods like turning into Badeline, no-backpack Variant, ...
The hair length sent to other players will be capped at 12 segments
(Madeline's default hair is 4 or 5 in length depending on dash count!)
Some popular hair mods and mods that specifically support CelesteNet include:

  • Hyperline - customize hair color and length per dash-count. Supports things like gradients, animated rainbow hair, ...
  • Prideline - lets you set your hair to LGBTQ+ pride flags!
  • Styline - allows one to mix various modifications and filters to adjust and recolor Madeline's sprite
  • Super Mario Bros. Skin
  • KyaruMod
  • ...and more? If you made or know of a mod that should be listed here, let us know!

#15 How do I find others to play with?

The easiest way to find others is to simply ask in global chat! Just suggest something like playing through 4B casually or beating Spring Collab Beginner maps and see if someone else will join in.
You can also join official Celeste Discord server and give yourself the @celesternetter role. You can ping this role in the #celestenet channel to find others to play with, e.g. for Madhunt.

#16 What was GhostNet?

A buggy mess. GhostNet has been superceded by CelesteNet.
